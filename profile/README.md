# ValGuide

ValGuide is an open-source platform for museums, cultural institutions, civic
spaces, campuses, and other visitor-facing organizations building digital
experiences on the web.

The main project combines a visitor app, curator-facing studio, admin tooling,
publishing workflows, QR and short-link entry points, and deployment scaffolding.
Tours and guides are part of the platform, alongside broader tools for
institutional content, operations, localization, and experimentation.

## Community And Contact

ValGuide is built for cultural institutions, contributors, operators, and
visitors working across different languages, disciplines, and levels of
technical experience.

Read the [Code of Conduct](https://github.com/valguide/.github/blob/main/CODE_OF_CONDUCT.md)
before participating in project spaces. For security issues, conduct reports, or
sensitive contact, email [support@valguide.com](mailto:support@valguide.com)
instead of opening a public issue.
